Tutorial 10: CSS

CSS stands for Cascading Style Sheets.
It essentially describes how HTML elements are to be displayed on screen, paper, or in other media.
CSS saves a lot of work, as t controls the layout of multiple web pages all at once.
There are 3 ways that CSS can be applied to HTML elements.

  • Inline- by using the style attribute in HTML elements.
  • Internal- by using style element in the head section.
  • External- by using an external CSS file.
        The most common way to add CSS, is to keep the styles in separate CSS files.

External

In the head section of the HTML file, use the link element and use rel with it equal to "stylesheet", and href element equal to the name of the file. More will be learned in the CSS tutorials.
