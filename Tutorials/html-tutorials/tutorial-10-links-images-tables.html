HTML Images

Target Attribute

Target specifies where the link is opened. It can have one of the following values.

  • _blank: Opens link doc in new tab
  • _self: Same window or tab
  • _parent: Opens in parents frame
  • _top: Opens in the full body of the window

Image tags can be used within the a tag such that the image is a link.
Using the button tag's onclick attribute, a button can also be a link.

Title Attribute & External Paths

Using the title attribute, hovering allows the user to be prompted.
External attribute allows the path to specified rather than the entire web address.

Images

Attributes

  • Alt: Describes the image
  • Style: Specify the width and height of an image rather than the width and height attributes: Use by-- width:size;height:size
  • src: Specify an image that is in another server.
  • float: Allows the image to sit left or right of text.
  • usemap: Control action depending on where image is clicked.

Map tag correlates to usemap attribute by its name attribute.
The area tag can be used within. Area uses coords, atttributes, shape, and href to ultimately specify the location.
