Target Attribute

Target specifies where the link is opened. It can have one of the following values.

  • _blank: Opens link doc in new tab
  • _self: Same window or tab
  • _parent: Opens in parents frame
  • _top: Opens in the full body of the window

Image tags can be used within the a tag such that the image is a link.
Using the button tag's onclick attribute, a button can also be a link.

Title Attribute & External Paths
