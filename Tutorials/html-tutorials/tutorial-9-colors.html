RGB Colors

Each value defines the intensity of red, green, and blue respectivley. 0 for all values is black, while 255 for all is white. Keeping the values the same will result in different shades of grey.

RGBA Colors

The A, or alpha if you will, is the transparency. 0.0 is fully transparent while 1.0 is not at all.

Example Right Here!

HEX Colors

Red is the first 2 values, and so on for green and blue. ff is the max value and 0 is the minimum.

HSL Colors

HSL stands for hue, saturation, and lightness. So, hue is the main color, saturation controls the sensitivity, and lightness is the deepness of the color.
in HSLA, the A is known as alpha. Alpha controls the transparency of the color.
