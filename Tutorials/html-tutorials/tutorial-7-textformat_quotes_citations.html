Tutorial 7: Text Formatting, Quotation/Citation Elements

Text Formatting Elements

Text has special elements to give special meaning.

  • b, bold text
  • strong, important text
  • i, italic
  • em, emphasized
  • mark, marked text
  • small, small text
  • del, deleted text
  • ins, inserted text
  • sub, subscript
  • sup, superscript

Some examples of unfamiliar text elements.
bold,Strong,italic emphasized, marked, small, del, ins, sub, sup

Quotation/Citation elements

Quotations are easy to use. It is denoted as 'q'.

Blockquotes are also an element and is denoted as its named. Using the cite attribute, blockquotes are used to state that it is quoted from a source.

The element abbr (abbreviation), is used to ... give useful information to browsers, translation systems, and search-engines.

Example: WHO was founded in 1948.

The address element defines contact info such as author or owener of a document or and article.

Example:

Written by John Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA

Citations elements are denoted as cite and defines the title of a work.

Example:
The Scream by Edvard Munch. Painted in 1893

Bi-directional Override element, bdo, overrides the current text direction.

Example of dir attribute within bdo tag:
This text will be written from right to left.
