Tutorial 12: Lists and Blocks

Two types of lists: Ordered and unordered.
Unordered list's list-style-type property:

  • disc: Sets the list item marker to a bullet
  • circle: sets the list item marker to a circle
  • square
  • none

For ordered list's type attribute:

  • type="1": Numbered
  • A: Uppercase letters
  • a: lowercase letters
  • I: numered with uppercase roman
  • i: lowercase roman
